# SCRIPT — YC Search Product Tour

**Target length:** ~45 seconds
**Target word count:** ~100 words at 2.2 wps
**Voice:** calm, dry, technical. Not hyped. Think a senior engineer walking a friend through a tool they built.

---

## Narration

Five thousand, eight hundred, fifty-one Y Combinator companies.

One search bar.

Type what you actually mean — "A I agents for healthcare workflows" — and it ranks by meaning, not just keywords.

Filter by batch, industry, or stage. The table updates on every keystroke.

Open the companion graph, and the whole market shows up as a 3D constellation. Every node is a company. Hover one to peek.

Click through — you get the snapshot, the tags, the forty nearest semantic neighbors, pulled straight from embeddings.

Y C Search. Every company, one query away.

## Beat Seeds (handed to storyboard)

1. **Hook** — the number + the search bar.
2. **Semantic search** — typing the natural-language query, results reranking.
3. **Filtering** — clicking a chip, count collapsing from 5,851 → 666.
4. **Companion graph** — the 3D constellation + hover tooltip w/ company image.
5. **Company profile** — detail page with snapshot + embedding neighborhood.
6. **Closer** — logo lockup, "Every company, one query away."
